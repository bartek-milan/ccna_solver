# ccna_solver
A browser-based tool that finds your annoying CCNA answers with image recognition.  

# Usage
Load the test on your screen, then fire up ccna_solver.  
Use the HOTKEYS to navigate.

# HOTKEYS
Num 1: Search for the answer.  
Num 2: show the correct answer on the screen.  
Press again to remove it.
Num 9: Closes up the program real fast.
